# Pug Testing

This project was bootstrapped with [Vite](https://vitejs.dev/guide/).

## Technology

Node v16.18.9
React
Material UI (MUI) v5
DayJS

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn dev
```

Go to http://localhost:5173

## Running the app production

```bash
$ docker compose up
```

Go to ${ip_address} (if you run in the local, go to http://localhost)
